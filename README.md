# Amazon Homepage UI Clone

An Amazon-inspired homepage UI built using HTML and CSS, replicating the layout and design of Amazon's landing page for front-end development practice.

## Features

* Amazon-style navigation bar.
* Hero section with promotional banner.
* Product category boxes and sections.
* Footer section similar to Amazon's homepage.
* Clean and visually appealing user interface.
* Well-structured and reusable HTML and CSS code.
* Static webpage design for front-end practice.

## Technologies Used

* HTML5
* CSS3

## Project Structure

```
Amazon-Homepage-UI/
│
├── index.html
├── style.css
├── images/
└── README.md
```

## How to Run the Project

1. Clone the repository:

```bash
git clone <your-repository-url>
```

2. Open the project folder in Visual Studio Code.

3. Open `index.html` in your browser or use the Live Server extension.
